# 1.neovim  
安装步骤:  
sudo add-apt-repository ppa:neovim-ppa/stable  
sudo apt-get update  
sudo apt-get install neovim  

配置文件路径:  
- '~/.config/nvim/init.lua'  
- 注意:without plugin  


# 2.starship  
配置文件路径:  
- '~/.config/starship.toml'  
- 在'~/.bashrc'文件中加入 'eval "$(starship init bash)"'  


# 3.lazygit
